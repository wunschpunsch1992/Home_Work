Программа является простым аналогом утилиты sudo в Unix. Работает только если пользователь является root.
Программа имеет следующий синтаксис: ./launch --uid=n --program=program_name arguments
  uid          - ключ, содержащий идентификатор пользователя, от имени которого хотим запустить program_name
  program_name - наименование программы, которую хотим запустить
  arguments    - аргументы, которые хотим передать program_name
  program      - ключ, содержащий наимеование program_name

Длинные ключи могут быть  заменены на короткие. Тогда синтаксис вызова имеет следующий вид:
  ./launch -u n -p program_name arguments

Программа не умеет работать с составными ключами, т.е. вызов ./launch -u 100 -p ls -la приведёт к аварийному завершению программы.
Зато программа справится с таким вызовом ./launch -u 100 -p  ls -l -a
Ключи uid и program можно менять местами.
Программа умеет запускать исполняемые файлы: ./launch -u 100 -p  ./hello_world

Если ключ uid некорректен:
  - содержит бесмысленное значение (например -u ololo) - программа выполняться не будет
  - содержит идентификатор пользователя, которого нет в системе - программа выполняться не будет
